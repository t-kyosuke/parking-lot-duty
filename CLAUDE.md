## ★当番割り当てアルゴリズム（最重要）

固定順リスト（11名）をポインタで循環的に回す。

```javascript
const COACH_ORDER = [
  '塚原匡祐', '国沢剛', '岸下和樹', '堀本和幸', '濱田広宣',
  '前里元樹', '林和憲', '松木正和', '橋戸佑介', '河井彩登', '大串洋尚'
];

function assignParking(practiceDays, attendance, startPointer) {
  let pointer = startPointer;
  const results = [];

  for (const day of practiceDays) {
    let assigned = null;

    for (let i = 0; i < COACH_ORDER.length; i++) {
      const idx = (pointer + i) % COACH_ORDER.length;
      const coach = COACH_ORDER[idx];

      if (attendance[day]?.[coach] === '◯') {
        assigned = coach;
        pointer = (idx + 1) % COACH_ORDER.length;
        break;
      }
    }

    results.push({ date: day, coach: assigned });
  }

  return { results, nextPointer: pointer };
}
```

## CSVパース仕様

### 調整さんCSVの形式
```csv
2025年SRS3年コーチ出欠確認
コーチ陣の出欠確認こちらでお願いします
日程,茂木隼人,橋戸佑介,外山岳,...
3/29(日) 関大ラグビーフェスティバル（仮）,◯,◯,△,...
4/4(土) 13:40-16:40,◯,◯,×,...
...
コメント,,,祝日、日曜日は...,...
```

### パースルール
- **先頭2行**：メタデータ → スキップ
- **3行目**：ヘッダー（コーチ名の列）
- **最終行「コメント」**：スキップ
- **空欄（`,,`）**：△（未定）と同等に扱う
- **名前照合**：スペースを除去して比較（「岸下 和樹」→「岸下和樹」）
- **同日複数行**：最初の行の出欠を使用
- **エンコーディング**：UTF-8（Shift_JISにもフォールバック対応）
- **日程列**から練習時間を自動抽出（例：`4/5(日) 13:00-14:20` → `13:00-14:20`）

## 対象日の判定ルール

### 自動判定
- **土曜日**（`(土)` を含む）→ 対象外
- **日曜日**（`(日)` を含む）→ 対象候補
- **祝日**（祝日リストに該当）→ 対象候補
- 「試合」「交流試合」「フェスティバル」キーワード → 試合（対象外）
- 「合宿」キーワード → 合宿（対象外）

### 種別（UIで変更可能）
- 「練習」（デフォルト）→ 駐車場当番が必要
- 「試合」→ 対象外
- 「合宿」→ 対象外
- 「休み」→ 対象外
- 「運動会等」→ 駐車場当番が必要

## UI設計

### 閲覧者モード（デフォルト画面）
- 当月の当番スケジュールをカード形式で表示
- **日付 + 曜日 + 練習時間 + 当番者名**を表示
- 月切り替え（◀▶ボタン）
- 当日の当番を「TODAY」バッジでハイライト
- 試合・合宿日はアイコン付きでグレーアウト
- データがない月は「まだ決まっていません」
- 画面下部に「🔒 管理者ログイン」ボタン

### 管理者モード（タブ切り替え）
#### タブ1：当番割り当て
1. 月選択ドロップダウン
2. CSVアップロード（ドラッグ&ドロップ対応）
3. 出欠プレビュー（◯/△/× タップ切り替え、日程種別・練習時間編集）
4. 「当番を決定する」ボタン
5. 割り当て結果表示（手動変更可能）
6. LINEアナウンス用コピーボタン
7. 累計担当回数（バーチャート）
8. 次回開始位置表示

#### タブ2：設定
- 次回開始位置の変更（ドロップダウン）
- 累計回数の手動調整（±ボタン）
- パスワード変更
- データのエクスポート/インポート（JSON）
- 全データリセット
